# Jogo da memória do pantanal com som

Um jogo educativo e interativo da memória com temática dos animais do Pantanal, combinando elementos visuais e sonoros para uma experiência imersiva de aprendizado.

## Decisões 
- cada animal é um objeto com nome, imagem e som.  
- criei uma tela inicial com os icones dos animais para que o usuario possa conhecer os sons antes de jogar 
- na tela inicial o jogador tem duas opções de dificuldade, normal e temporizada.
- para fazer o embaralhamento das cartas foi utilizado o algoritmo Fisher-Yates.


## Fontes

- **Imagens**: [Pixabay](https://pixabay.com/pt/)
- **Áudios**: [Xeno-canto](https://xeno-canto.org/) e [YouTube](https://www.youtube.com/)

## Como Jogar

1. Abra `index.html` no navegador
2. Clique em "Iniciar Jogo Normal" ou "Iniciar Jogo por Tempo"
3. Clique nas cartas para revelar imagens e sons
4. Encontre pares combinando imagem com som
5. Maximize sua pontuação!
